# ADR-004 – SCSS Modules Choice

## Context
Deciding the styling method for the frontend. Options: Tailwind, Bootstrap, or SCSS Modules.

## Decision
We will use **SCSS Modules**.

## Justification
- Allows local scope per component
- More control over design
- Facilitates learning modern CSS best practices

## Consequences
- ✅ Modular and reusable code
- ❌ Requires attention to class naming

# ADR-004 – Escolha do SCSS Modules no front (PT)

## Contexto
Decidir a forma de estilização no frontend. Opções: Tailwind, Bootstrap ou SCSS Modules.

## Decisão
Usaremos **SCSS Modules**.

## Justificativa
- Permite escopo local por componente
- Mais controle de design
- Facilita aprendizado de boas práticas de CSS moderno

## Consequências
- ✅ Código modular e reutilizável
- ❌ Requer atenção na nomenclatura de classes
