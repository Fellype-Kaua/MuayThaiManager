# ADR-005 – Naming Conventions

## Context
Define naming conventions for files, folders, variables, and functions.

## Decision
- Files and folders: **kebab-case**
- React components: **PascalCase**
- Variables and functions: **camelCase**
- SCSS classes: **camelCase** inside modules

## Justification
- Consistency and readability
- Easier understanding for other developers and for portfolio

## Consequences
- ✅ Standardized and maintainable code
- ❌ Must strictly follow the rules from the start

# ADR-005 – Nomenclaturas (PT)

## Contexto
Definir padrão de nomenclatura de arquivos, pastas, variáveis e funções.

## Decisão
- Arquivos e pastas: **kebab-case**
- Componentes React: **PascalCase**
- Variáveis e funções: **camelCase**
- Classes SCSS: **camelCase** dentro de módulos

## Justificativa
- Consistência e legibilidade
- Facilita entendimento para outros devs e portfólio

## Consequências
- ✅ Código padronizado e fácil de manter
- ❌ É necessário seguir regra rigorosa desde o início
