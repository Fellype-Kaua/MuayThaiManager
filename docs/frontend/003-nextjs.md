# ADR-003 – No Next.js Choice

## Context
Decide whether the frontend should use Next.js or React (Vite).

## Decision
We will use **React + Vite**, without Next.js.

## Justification
- Internal project, no need for server-side rendering  
- Less complexity for the MVP  
- Faster and easier development

## Consequences
- ✅ Faster development  
- ❌ Some SEO and SSR features unavailable (not needed for MVP)

# ADR-003 – Escolha de não usar Next.js (PT)

## Contexto
Decidir se o frontend usaria Next.js ou React puro (Vite).

## Decisão
Usaremos **React + Vite**, sem Next.js.

## Justificativa
- Projeto interno, não precisa de renderização server-side  
- Menos complexidade para o MVP  
- Mais fácil e rápido de desenvolver

## Consequências
- ✅ Desenvolvimento mais ágil  
- ❌ Algumas funcionalidades de SEO e SSR não estarão disponíveis (não necessárias para MVP)
