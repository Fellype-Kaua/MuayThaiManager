# ADR-003 – Project Patterns

## Context
Decide project patterns for backend and frontend.

## Decision
- Backend: MVC (Model, Controller, Route)  
- Frontend: Functional React components with hooks

## Justification
- Easier maintenance and testing  
- Follows modern best practices

## Consequences
- ✅ Easier understanding for other developers  
- ❌ Requires discipline to not break the pattern

# ADR-003 – Padrões de projeto (PT)

## Contexto
Decidir padrões de projeto para backend e frontend.

## Decisão

- Frontend: componentes funcionais React com hooks

## Justificativa
- Facilita manutenção e testes  
- Segue boas práticas modernas

## Consequências
